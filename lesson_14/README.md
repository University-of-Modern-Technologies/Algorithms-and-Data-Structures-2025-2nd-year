# Двійкове дерево пошуку (Binary Search Tree)

Цей проект реалізує двійкове дерево пошуку (BST) з можливістю вставки, видалення та пошуку елементів. Включає візуалізацію дерева та різні види обходу.

## Структура проекту

- `bst.py` - Реалізація BST з функціями вставки, пошуку та видалення
- `main.py` - Основна логіка візуалізації дерева за допомогою NetworkX
- `order.py` - Реалізація різних видів обходу дерева (preorder, inorder, postorder)
- `bst_visualizer/` - Інтерактивна візуалізація BST у веб-браузері
- `order_visual/` - Візуалізація обходів дерева

## Основні функції

### Функції BST

- `insert(root, key)` - Вставка нового елемента
- `search(root, key)` - Пошук елемента
- `delete(root, key)` - Видалення елемента
- `min_value_node(node)` - Знаходження мінімального значення в піддереві

### Обходи дерева

- **Inorder** (Симетричний обхід) - відсортоване подання
- **Preorder** (Прямий обхід) - корінь → ліво → право  
- **Postorder** (Зворотний обхід) - ліво → право → корінь

## Технічна реалізація

### Основні класи та функції

#### Node

Клас, що представляє вузол дерева:

- `left` - лівий дочірній вузол
- `right` - правий дочірній вузол
- `val` - значення вузла
- `color` - колір вузла (для візуалізації)
- `id` - унікальний ідентифікатор

#### Основні функції BST

- `insert(root, key)` - рекурсивна вставка елемента
- `search(root, key)` - рекурсивний пошук елемента
- `delete(root, key)` - видалення елемента з обробкою різних випадків
- `min_value_node(node)` - знаходження мінімального значення

#### Візуалізація

- `add_edges(graph, node, pos, x=0, y=0, layer=1)` - додавання ребер до графа для візуалізації
- `draw_tree(tree_root)` - відображення дерева за допомогою matplotlib
