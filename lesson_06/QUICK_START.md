# Швидкий старт

## Запуск застосунку

```bash
python main.py
```

Оберіть алгоритм (1-3):

1. Bubble Sort
2. Insertion Sort
3. Selection Sort

## Керування

У вікні візуалізації:

- **Next** → наступний крок
- **Reset** → початковий стан
- Закрити вікно → вихід

## Приклад

```bash
> python main.py
Виберіть алгоритм сортування:
1 - Bubble Sort
2 - Insertion Sort
3 - Selection Sort

Початковий масив: [5, 3, 8, 4, 2, 7, 1, 6]
Кількість кроків: 14

Запуск візуалізації Bubble Sort...
```
