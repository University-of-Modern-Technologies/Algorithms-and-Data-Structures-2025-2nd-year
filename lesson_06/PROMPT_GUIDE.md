# Промпт для створення застосунку візуалізації сортування

## Оригінальний запит

Якщо б ви хотіли створити подібний застосунок з нуля, ось ефективний промпт для AI асистента:

---

**Промпт:**

```
Створи застосунок для покрокової візуалізації алгоритмів сортування на Python.

Вимоги:
1. Алгоритми: Bubble Sort, Insertion Sort, Selection Sort
2. Візуалізація: matplotlib з bar chart
3. Формат: покрокове відображення (кнопки Next/Reset)
4. Елементи керування: мінімальні (тільки Next та Reset)

Структура:
- Окремі файли для кожного алгоритму
- Модуль візуалізації з класом
- Головний файл з вибором алгоритму

Кожен алгоритм повинен мати функцію *_steps() що повертає список станів масиву.
```

---

## Що було реалізовано

### 1. Алгоритми з генерацією кроків

- `bubble.py` - Bubble Sort + bubble_sort_steps()
- `insertion.py` - Insertion Sort + insertion_sort_steps()
- `selection.py` - Selection Sort + selection_sort_steps()

### 2. Модуль візуалізації

- `visualizer.py` - клас StepVisualizer
  - matplotlib bar chart
  - Кнопки Next/Reset
  - Автоматичне оновлення графіка

### 3. Головний файл

- `main.py` - інтерфейс вибору алгоритму
- Консольне меню 1-3
- Запуск обраної візуалізації

### 4. Допоміжні файли

- `README.md` - повна документація
- `test_algorithms.py` - автотести
- `test_visualizer.py` - тестування візуалізації

## Результати тестування

Всі алгоритми пройшли 7 тестових випадків:

- Базовий масив
- Вже відсортований
- Зворотний порядок
- Один елемент
- Однакові елементи
- Великий масив
- Два елементи

**Кількість кроків для [5, 3, 8, 4, 2]:**

- Bubble Sort: 8 кроків
- Insertion Sort: 12 кроків
- Selection Sort: 4 кроки

## Ключові рішення

1. **Генерація кроків** - кожна функція \*\_steps() зберігає копію масиву після кожного обміну
2. **Покрокова візуалізація** - StepVisualizer зберігає поточний крок та оновлює графік
3. **matplotlib.widgets.Button** - для інтерактивних кнопок
4. **Мінімальний інтерфейс** - лише Next/Reset без зайвих функцій

## Як використовувати

```bash
python main.py
# Оберіть алгоритм 1-3
# Натискайте Next для перегляду кроків
# Reset для повернення на початок
```
