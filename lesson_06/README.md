## Алгоритми

### Bubble Sort

- Складність: O(n²)
- Порівнює сусідні елементи та міняє їх місцями якщо вони в неправильному порядку
- Найбільший елемент "спливає" в кінець масиву

### Insertion Sort

- Складність: O(n²)
- Будує відсортований масив по одному елементу за раз
- Вставляє кожен новий елемент у правильну позицію

### Selection Sort

- Складність: O(n²)
- Знаходить мінімальний елемент та переміщує його на початок
- Повторює процес для решти масиву

## Приклади

Масив `[5, 3, 8, 4, 2]`:

- Bubble Sort: 8 кроків
- Insertion Sort: 12 кроків
- Selection Sort: 4 кроки

## Особливості візуалізації

- **Активні елементи** виділяються червоно-помаранчевим кольором
- **Bubble Sort** показує елементи що порівнюються і обмінюються
- **Insertion Sort** показує елемент що вставляється і куди
- **Selection Sort** показує поточний мінімум і елемент що перевіряється
- В заголовку відображаються індекси активних елементів
