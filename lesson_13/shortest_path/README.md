# Пошук найкоротших шляхів

У цій директорії зібрані приклади пошуку найкоротших шляхів у графах за допомогою бібліотеки NetworkX.

## Основні інструменти

### 1. `nx.shortest_path`
Універсальна функція, яка автоматично обирає алгоритм:
- Для **незважених графів** використовує пошук у ширину (BFS), оскільки найкоротший шлях вимірюється кількістю ребер.
- Для **зважених графів** (з аргументом `weight`) використовує алгоритм Дейкстри.

### 2. `bfs_shortest_path.py` (Ручна реалізація)
Приклад того, як працює пошук найкоротшого шляху "під капотом" для незваженого графа. 
- Використовує чергу `deque`.
- Гарантує знаходження шляху з мінімальною кількістю кроків.

### 3. Візуалізація
Файл `visualize_graph.py` показує граф та підсвічує знайдений шлях червоним кольором.
