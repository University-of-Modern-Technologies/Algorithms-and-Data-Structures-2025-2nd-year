# Умова

У багатьох мовах програмування ми маємо справу з виразами, виділеними символами-розмежувачами,
такими як круглі дужки `( )`, квадратні дужки `[ ]` або фігурні дужки `{ }`.

Напишіть програму, яка читає послідовність символів-розмежувачів, таку як `( ) { [ ] ( ) ( ) { } } }`, і повідомляє,
коли розмежувачі симетричні або коли несиметричні, наприклад `( ( ( )`; або коли розмежувачі різних видів стоять у парі,
як `( }`. Використовуйте стек, щоб запам'ятати відкриті в даний момент символи-розмежувачі.
