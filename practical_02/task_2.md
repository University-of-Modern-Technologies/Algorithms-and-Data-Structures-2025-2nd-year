# Завдання 3 “Об'єднання k відсортованих списків”

Дано `k` відсортованих списків цілих чисел. Ваше завдання — об'єднати їх у один відсортований список.
При виконанні завдання можете опиратися на алгоритм сортування злиттям з конспекту. Реалізуйте функцію `merge_k_lists` , яка приймає на вхід список відсортованих списків та повертає відсортований список.

Приклад очікуваного результату:

```python
lists = [[1, 4, 5], [1, 3, 4], [2, 6]]
merged_list = merge_k_lists(lists)
print("Відсортований список:", merged_list)
```

Виведення:

```python
Відсортований список: [1, 1, 2, 3, 4, 4, 5, 6]
```
