# Тема 2. Основні структури даних

## Опис заняття
Ключові структури для зберігання й обробки даних. Масиви. Списки. Словники. Множини. Стек. Черги. Зв'язні структури даних: Граф. Дерева.

## Файли заняття

### `calculator.py`
Основний файл з реалізацією калькулятора для демонстрації базових структур даних та операцій з ними.

### `consumer.py`
Реалізація consumer (споживача) для черги RabbitMQ - отримує та обробляє повідомлення з черги 'task_queue'.

### `example.ipynb`
Jupyter ноутбук з прикладами та поясненнями основних структур даних, включаючи практичні завдання та візуалізацію.

### `graph.py`
Базова демонстрація створення графу за допомогою NetworkX: додавання вузлів, ребер та візуалізація графу.

### `producer.py`
Реалізація producer (виробника) для черги RabbitMQ - надсилає повідомлення у чергу 'task_queue' через exchange.

## Рекомендації для студентів
1. Почніть з перегляду `example.ipynb` для розуміння теоретичних основ
2. Вивчіть `calculator.py` для розуміння практичного застосування структур даних
3. Проаналізуйте роботу з чергами через `producer.py` та `consumer.py`
4. Розгляньте `graph.py` для розуміння графових структур
